05 · Architecture

Under the Hood

DeepSeek V4 Pro via OpenRouter on Nous Research's Hermes Agent framework. Linux native, fully extensible.

🧬

Foundation Model

DeepSeek V4 Pro delivers state-of-the-art reasoning, coding, and creative generation — accessed via OpenRouter for reliability and speed.

⚙️

Hermes Agent Framework

Nous Research's open-source platform: tool-use, persistent memory, skills system, cron scheduling, and multi-platform delivery.

🧩

Skill Ecosystem

74 hot-loadable skills spanning Twitter/X, GitHub, smart contracts, NFT analysis, ML tooling, creative generation, and more.

🐧

Linux-Native

Runs on Ubuntu with full terminal access, background processes, desktop control, and Docker support — real OS integration, not a sandbox.

# JARVIS Agent Architecture
from hermes_agent import Agent

jarvis = Agent(
    model="deepseek/deepseek-v4-pro",
    provider="openrouter",
    tools=["terminal","browser","computer_use",
           "web_search","code_exec","image_gen",
           "cron","xurl","memory","delegate",
           "voice_tts","process"],
    skills="autoload",
    memory=True,
    creator="@kubiyaro"
)

jarvis.run("Update my site, roast my enemies, find alpha")
